# Curso Interactivo: Bases de Datos, PHP, y APIs

## Bienvenido/a

Este curso está diseñado para alguien que parte desde cero. Vamos paso a paso,
con teoría clara, ejemplos que puedes ejecutar y ejercicios para practicar.

---

## Estructura del Curso

| Módulo | Tema | Descripción |
|--------|------|-------------|
| 1 | Bases de Datos | Qué son, para qué sirven, SQL desde cero |
| 2 | PHP Básico | Variables, funciones, estructuras de control |
| 3 | PHP + Base de Datos | Conectar PHP con MySQL, operaciones CRUD |
| 4 | Endpoints y APIs | Qué es HTTP, cómo crear endpoints en PHP |
| 5 | Proyecto Final | Sistema completo: API + Base de datos |

---

## Cómo usar este curso

1. **Lee** cada archivo de teoría (`teoria.md`) en orden
2. **Estudia** los ejemplos en la carpeta `ejemplos/`
3. **Resuelve** los ejercicios en la carpeta `ejercicios/`
4. **Compara** tus soluciones con las que se proporcionan

---

## Requisitos para ejecutar los ejemplos

- PHP 7.4 o superior
- MySQL o MariaDB
- Un editor de texto (como Cursor)
- Un navegador web
